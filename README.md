# neural-network-C

基于C语言的神经网络框架,目前包含全连接层。支持前向传播推理和反向传播训练。使用tensor实现各种功能，适合人工智能入门学习。

## 各文件的主要功能

tensor.c		:	tensor的各种基本运算，包括创建，删除tensor，矩阵乘法等。

net.c		 	:	神经网络的主体，包含网络的创建，添加层，前向，反向传播相关函数，优化器函数等。

compute.c	:	包含更加复杂的数学计算，包括损失函数，激活函数以及他们的导函数。
